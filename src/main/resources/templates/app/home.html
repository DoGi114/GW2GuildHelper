This website was created due to Guild Wars 2 API limitation to not holding every action in it. As for this app it holds every API entry since creation of the guild in it. It refreshed guild every 10 minutes or on clicking "Refresh data" button so you can have all the API entries. You can also easily iterate and search through items, logs, donations and roaster.

Admin profile is the one who creates the guild 1st. You can also define other admins in Settings section. Application requires guild leader API key and Guild ID. If you dont know your guild ID refer to : wiki
